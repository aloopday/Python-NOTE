赫尔辛基大学免费MOOC课程的最新列表
嘿，我是在这所大学读书的学生，想大致了解一下我们免费向所有人开放的课程，所以我就想“为什么不在这里发帖，就像 https://www.reddit.com/r/learnprogramming/comments/m8nmuu/all_the_mooc_of_helsinki_university/ 那样”？ 那么，开始吧：

Python 编程 MOOC (14 周)： https://programming-23.mooc.fi/

这门课相当于我们的编程 I 和编程 II。它涵盖了所有基础知识，从变量和 JSON 到更高级的概念，比如 OOP 和函数式编程风格的基础知识，最后甚至还有一点用 Python 做的游戏开发。

Java 编程 MOOC (14 周)： https://java-programming.mooc.fi/

和 Python 的那个很像，只不过是用 Java 做的。我自己没学过，所以没法多评论。如果你感兴趣，可以浏览一下内容。

Haskell MOOC (16 节课)： https://haskell.mooc.fi/

这门课是学习 Haskell 和学习通用函数式编程原则的结合。第一部分有 127 个练习，第二部分有 121 个，但我的经验是，它不会太重复（每个练习都非常清楚地说明了一个新概念）。

Full Stack Open (14 部分)： https://fullstackopen.com/en/

这是一门涵盖网络开发的大型课程，内容相当丰富。参与其中的公司也承诺，那些完全完成课程内容的人可以在面试时获得机会！（尽管他们几乎只在芬兰运营）

用 Python 进行数据分析 (7 部分)： https://courses.mooc.fi/org/uh-cs/courses/dap-22

正如标题所说，它教你用 Python 进行数据分析的基础知识。有一些数学先决条件，所以如果你不了解基本的线性代数和微积分，可能就太难了。

网络安全 (14 部分)： https://cybersecuritybase.mooc.fi/

这门课通过作业和个人小型项目，涵盖了许多网络安全主题的基础知识。例如：

“在第二个项目中，参与者将安装一个具有各种漏洞的操作系统，然后在其上安装一个网络入侵防御系统。随后，参与者将熟悉渗透测试软件，并攻击他们安装的系统。”

人工智能基础：(2 部分) https://www.elementsofai.com/

从非计算机科学家的角度简单介绍人工智能的基础知识。只需要基本的 Python，而且只在第二部分用到。

社会中的人工智能 (3 部分) https://courses.mooc.fi/org/uh-cs/courses/ai-in-society

对于那些想了解人工智能如何与社会互动的人来说，这是一个更柔和的视角。

5G MOOC (4 部分) https://courses.mooc.fi/org/uh-cs/courses/5g-mooc

类似于人工智能基础对人工智能的介绍，介绍了 5G 技术。涵盖了 SIM 卡和网络基础知识等内容

CSES 问题集： https://cses.fi/problemset/list/

大约 300 个非常难的算法作业。
