 F- String was introduced in Python3.6, and is now the preferred way of formatting strings.
 To specify a string as an f-string, simply put an `f` in front of the string literal, and 
 add curly brackets `{}` as placeholders for variables and other operations.

 ## Placeholders and Modifiers
  A placeholder can contain variabes, operations, functions and modifers to format the value.
